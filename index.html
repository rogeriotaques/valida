Valida (jQuery) is a client side form validation plugin written by Rogério Taques. It provides an easy, fast and lightweight way to make form validations on client-side. Moreover, it's source-code (minified) has about only 12Kb.

Valida works really well with Bootstrap, but you can also use it along any other CSS framework or boilerplate available in the market. Have you tried Seed CSS (the one I've made)?
